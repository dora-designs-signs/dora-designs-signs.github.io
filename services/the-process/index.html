The Process

If it can be cut, we can cut it!

However, some materials are better suited for certain cutting processes - if you are unsure of the best way, just ask us! Some materials, such as Polycarbonate, cannot be lasered due to harmful fumes released when burned.

What is required?

To receive a quote you must submit a drawing (.pdf), 3D model (.stp, .igs, .xt, etc.), or vector file (.svg, .dxf, .ai, etc.) If you need specific tolerances or fitments please provide a drawing so we can ensure a perfect part every time.

If you do not have any files to submit, images must be sent along with approximate dimensions and a description of work to be done.

You will recieve a follow-up email or call within 24hrs with any questions.

-Note- CNC machines require programming, and programming requires digital features (models or vectors) to define movements of the machine. Design costs will be added if digital files are not supplied and can vary greatly depending on the complexity of the work.
